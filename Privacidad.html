Política de Privacidad

Última actualización: junio de 2026

Esta página explica de forma clara qué datos personales se recogen a través del sitio web de Jota Fix (en adelante, "el sitio web"), con qué finalidad se utilizan y qué derechos tienes al respecto. El objetivo es que sepas en todo momento qué ocurre con tu información si decides escribirnos o usar el formulario de contacto.

1. Quién trata tus datos

Jota Fix es un proyecto personal de soporte técnico y mantenimiento de equipos informáticos, gestionado por Joaquín. Puedes ponerte en contacto sobre cualquier cuestión relacionada con tus datos a través de los canales indicados en la sección "Contacto" de esta misma política.

2. Qué datos se recogen

A través de este sitio web únicamente se recogen los datos que tú mismo decides facilitar de forma voluntaria, concretamente al rellenar el formulario de contacto:

  • Nombre
  • Correo electrónico
  • Teléfono (opcional)
  • Servicio de interés
  • El mensaje que escribas

El sitio web no utiliza cookies de seguimiento ni herramientas de analítica o publicidad, por lo que no se recopila información de navegación con fines estadísticos ni publicitarios.

3. Cómo se utilizan estos datos

El formulario de contacto de este sitio web no envía tus datos a ningún servidor ni base de datos. Al pulsar "Enviar por WhatsApp", tu propio navegador genera un mensaje con la información introducida y lo abre directamente en WhatsApp, para que seas tú quien decide enviarlo. A partir de ese momento, la conversación se gestiona como cualquier chat normal de WhatsApp, entre tú y Jota Fix.

Los datos que nos facilites por este medio (o directamente por WhatsApp, Instagram o teléfono) se usan exclusivamente para responder a tu consulta, gestionar la solicitud de un servicio y, si procede, coordinar la cita o recogida del equipo. No se utilizan para ningún otro fin, ni se ceden a terceros con fines comerciales.

4. Base legal

El tratamiento de tus datos se basa en el consentimiento que prestas voluntariamente al rellenar el formulario o al iniciar una conversación por WhatsApp, Instagram o cualquier otro canal de contacto, así como en la necesidad de gestionar la relación previa a la prestación del servicio que solicitas.

5. Cuánto tiempo se conservan

Al no existir una base de datos del sitio web, no se almacena ningún dato más allá del propio historial de tu chat de WhatsApp o Instagram (que conservas tú igual que nosotros, según las condiciones de esas plataformas). Los datos de contacto que pudiéramos guardar de forma manual (por ejemplo, tu número de teléfono en la agenda, para futuras citas) se conservan únicamente mientras exista una relación de servicio o hasta que solicites su eliminación.

6. Servicios de terceros utilizados

Este sitio web utiliza los siguientes servicios externos para su funcionamiento, que pueden procesar datos técnicos básicos (como tu dirección IP) según sus propias políticas de privacidad:

  • WhatsApp (Meta) — para la gestión de las conversaciones iniciadas desde el formulario o el botón de contacto.
  • Google Fonts — para la carga de las tipografías del sitio.
  • Font Awesome / Tailwind CSS (CDN) — para iconos y estilos visuales.
  • OpenStreetMap / CARTO (vía Leaflet) — para mostrar el mapa de ubicación.

Ninguno de estos servicios recibe los datos que escribes en el formulario de contacto; solo cargan recursos técnicos (tipografías, iconos, mapas) necesarios para visualizar la página.

7. Tus derechos

En relación con cualquier dato personal que nos hayas facilitado, puedes en cualquier momento:

  • Solicitar acceso a los datos que tengamos sobre ti.
  • Pedir que se corrijan si son inexactos.
  • Solicitar su eliminación (por ejemplo, borrar tu número de un chat o agenda).
  • Oponerte al tratamiento o retirar tu consentimiento.

Para ejercer cualquiera de estos derechos, basta con que nos lo indiques a través de WhatsApp o Instagram (ver sección de contacto). Atenderemos tu solicitud a la mayor brevedad posible.

8. Seguridad

Se aplican medidas razonables para evitar el acceso no autorizado, la pérdida o la alteración de los datos que pudieran conservarse (por ejemplo, protección del propio dispositivo y cuentas utilizadas para la actividad). Dado que no existe una base de datos online asociada a este sitio web, el principal "almacenamiento" de tus datos de contacto es el chat de la app de mensajería que tú mismo utilices.

9. Menores de edad

Los servicios ofrecidos están dirigidos a un público general. Si eres menor de edad, te recomendamos contar con la supervisión de un adulto responsable antes de facilitar tus datos o solicitar un servicio.

10. Cambios en esta política

Esta política puede actualizarse para reflejar cambios en el funcionamiento del sitio web o en la normativa aplicable. La fecha de la última actualización aparece al principio de esta página. Te recomendamos consultarla de vez en cuando si tienes dudas sobre el tratamiento de tus datos.
